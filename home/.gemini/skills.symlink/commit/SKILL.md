# Git commit

- Git のコミットメッセージを生成してください
- 下記のコマンドを用いて現在の変更内容や最近のコミットメッセージを把握のうえで、適切なコミットメッセージを生成してください
    - `git status`
    - `git diff --cached`
    - `git log -n 3`
- プロジェクトでコミットメッセージにルールがあるなら、それに準拠してください
- もしステージングされているファイルが無い場合、ダーティな変更があるならそれをステージングするかをユーザーに問い合わせてください
    - ダーティな変更もないならユーザーのその旨を回答して終了してください
- メッセージの生成後、その内容でコミットしてください
- コミット時にフックによるエラーが発生した場合は、`-n` オプションを付けて再実行するかどうかをユーザーに問い合わせてください
