---
name: rename-feat-branch
description: 現在の Git ブランチをコミットログおよび差分の内容に基づいて機能ブランチにリネームする
---

- `git log '@{u}'...@ -p` の内容に基づいて変更内容に相応しい機能ブランチの名前を考えてください
- 機能ブランチ名の命名規則は下記のコマンドなどで最近のコミットメッセージや既存のブランチ名から推測してください
    - `git log -n 20 --oneline --graph`
    - `git branch`
- もしくは、プロジェクトで機能ブランチ名にルールがある場合、それに準拠してください
- 機能ブランチ名が決まったら、ユーザーにそれでいいかを問い合わせてください
- ユーザーが承諾したら `git branch --move` で現在のブランチを新しい機能ブランチにリネームしてください
